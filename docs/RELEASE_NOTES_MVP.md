# RELEASE_NOTES_MVP

## Что нового в MVP “Weekly Autopilot”

- Добавлена генерация недельного плана в one-tap сценарии.
- Добавлена быстрая замена приема пищи с умными вариантами.
- Добавлена поддержка повторов meal-prep на несколько дней.
- Улучшен список покупок: теперь с количествами и более понятной структурой.
- Добавлена мягкая адаптация плана после “съел не по плану / поел вне дома”.
- Усилен бюджетный контур: strict/soft и режим «сделать дешевле».
- Улучшено использование инвентаря и продуктов с коротким сроком.
- Добавлены более прозрачные объяснения “Почему?” для рекомендаций.

## Что важно знать

- Если часть данных недоступна (цены/единицы/health), приложение продолжает работать в best-effort режиме и сообщает, где оценка приблизительная.
- Тон уведомлений остается мягким и без давления.
