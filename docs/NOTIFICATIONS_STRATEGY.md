# NOTIFICATIONS_STRATEGY

## Принципы

- Только мягкий тон.
- Без стыда, без давления.
- Пользователь управляет категориями уведомлений.
- Учитываем quiet hours.

## Обязательные категории

1. Expiring soon
- Триггер: продукт близок к сроку годности.
- Пример текста: «Сегодня лучше использовать: йогурт и шпинат. Могу предложить 3 быстрых блюда.»

2. Shopping reminder
- Триггер: у пользователя есть незакрытый список покупок перед планом дня/недели.
- Пример текста: «Для завтрашнего плана не хватает 4 позиций. Открыть список покупок?»

## Опционально

3. Meal time reminder (gentle)
- Триггер: пользователь включил напоминания по времени.
- Пример текста: «Время обеда. У тебя уже готов простой вариант на сегодня.»

## Каналы и настройки

- Включение/выключение по категориям.
- Quiet hours с диапазоном времени.
- Локализация RU.
