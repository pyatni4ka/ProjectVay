# Домашний Инвентарь + ИИ-Рецепты + План Питания

Монорепозиторий MVP для iPhone-приложения (SwiftUI) и дешёвого backend-сервиса индексации/поиска рецептов.

## Структура

- `ios/` — нативное iOS-приложение (SwiftUI, HealthKit, Notifications, VisionKit).
- `backend/` — Node.js/TypeScript API для рецептов и (опционально) barcode proxy.
- `docs/` — архитектура, API-спецификация, формула ранжирования, тест-план.

## Быстрый обзор функциональности MVP

1. Инвентарь дома: холодильник, морозилка, шкаф; партии и сроки годности.
2. Сканирование EAN-13/DataMatrix с локальным кэшем и fallback на ручное создание.
3. Уведомления о сроках 5/3/1 день с учётом тихих часов.
4. Рекомендации рецептов с приоритетом “скоро испортится”, учётом похудения и бюджета.
5. План питания 3 приёма пищи/день + учёт напитков и кофе.
6. Прогресс по данным Apple Health и мягкая корректировка цели калорий.

Подробности см. в `docs/ARCHITECTURE.md` и `docs/API_SPEC.md`.
